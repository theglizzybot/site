Frequently Asked Questions

How do I install Glizzy?

Visit our GitHub repository for detailed installation instructions. The README contains step-by-step setup guides.

What are the system requirements?

Glizzy requires Node.js 18+ and a Discord bot token. See the Configuration Guide for more details.

Can I customize the bot messages?

Yes! Glizzy is fully configurable. Check the Configuration Guide for available customization options.

How do I invite the bot to my server?

You can deploy your own instance using the code from our GitHub repository, or join our support server for more information.

Is there support available?

Yes! Join our Discord support server for help and community support.

How often are updates released?

We regularly release updates and new features based on community feedback. Check GitHub for the latest releases.
