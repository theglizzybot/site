About Glizzy

Glizzy is a Discord bot designed to seamlessly integrate your Minecraft server with Discord. It relays in-game chat to Discord channels, posts player join/leave and server status updates, and exposes convenient Discord commands for server management.

Built with flexibility and ease of use in mind, Glizzy supports automated announcements, role-aware notifications, and customizable message formatting so server owners can tailor behavior to their community needs.

Whether you're running a small survival server or a large community network, Glizzy helps streamline communication between players and moderators while keeping server activity visible in Discord.
